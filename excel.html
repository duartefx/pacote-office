EXCEL

Uma ferramenta do Pacote Microsoft Office

Excel ou Microsoft Excel é um aplicativo de criação de planilhas eletrônicas. Foi criado pela Microsoft em 1987 para computadores que usam o sistema operacional da empresa antes já rodava no Mac (1985). O Excel é a melhor plataforma eletrônica para criação de planilhas. Muitas empresas hoje sobrevivem com base em uso destas planilhas.

FUNÇÕES:

  • O Excel oferece uma variedade de estilos de formas de gráficos que podem ser veiculadas em outros arquivos, como documentos de texto ou slides.
  • São uma série de fórmulas matemáticas pré-desenhadas para gerar resultados de forma rápida e eficaz.
  • É possível organizar desde o orçamento financeiro familiar, o fluxo de caixa de uma loja de materiais de construção, calcular as horas trabalhadas dos funcionários de uma pequena empresa, organizar os dados de vendas e diversas outras possibilidades.
